Coding for Fun!

Why I Know Programming is For Me

Even in my free time, between work, schooling and codecademy courses, I still find myself working on personal development projects. Whether it's updating this portfolio website, learning a new language, a new framework or building a new application, I simply love to create. Programming gives me the tools to endlessly create but also benefit from the hard work of others before me. My programming professor in high school gave a quote I love to this day, "Good programmers write good code. Great programmers borrow great code!" It's perfectly said, there is so much widely available to learn from and experiment with. My love for continual self-improvement and personal projects helps me keep up to date with current technologies.

Budgit

Budgit is a React budgeting app that takes incomes and expenses and determines your weekly and monthly budgets. Budgit will incorporate living expenses and savings calculations to make sure you are maximizing on your income. Currently is still a work in progress. In the future, I would like to incorporate Plaid API for connecting bank accounts to the app so that the budgeting calculations are fully automated. Currently using Git for version control.
