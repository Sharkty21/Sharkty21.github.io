Ravenous

Ravenous is a React App made in Codecademy's Web Development Path. The project is essentially a Front-End restaurant search app utilizing Yelp API. The user can search restaurants based on location and sort by best match, highest rated and most reviewed. The app utilizes React Components, state passes, fetch requests and json responses from the Yelp API. I deployed the app on Netlify as well to practice CI/CD. Jammming

Jammming is a React App made in Codecademy's Web Development path. Jammming allows users to create and edit playlists to their Spotify account. Jammming uses an authentication key through Spotify's API to access and edit the user's account. Jammming uses React components to search Spotify songs, add them to a custom playlist, then push to the user's account. I used surge to deploy this project. To view a live demo of the code click the image or check out my Github for the code.

Expresso

Expresso is a React App made in Codecademy's Web Development path. It is used to manage menus and employees in a restauraunt. Expresso uses a Full Back-End CRUD API and sqlite3 database to access and edit data information. Specifically, users can utilize the api.js with multiple Express routers in order to make GET, POST, PUT and DELETE requests to the Express server.js in order to manipulate sqlite3 database. To view code and run program, check out my Github.
